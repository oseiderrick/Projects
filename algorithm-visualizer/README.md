Algorithm Visualizer (BFS/DFS/Dijkstra/Sorts)

Overview
Interactive visualizer implemented in Python with Pygame. Step controls and basic complexity cues.

Run
1) Python 3.11+
2) pip install -r requirements.txt
3) python visualizer.py

Controls
- Space: step/advance
- B: BFS
- D: DFS
- K: Dijkstra
- R: reset
